anaDiff_Agilent {tools}R Documentation

Differential analysis for Agilent array

Description

Comparison analysis using limma.

This fonction read the summary file and execute the several comparison of it. Library used are limma, httr and jsonlite.

Usage

anaDiff_Agilent(designPuce)

Arguments

designPuce

character or integer. The Agilent number of the design

labelling

character. The default is "direct" for a direct labelling of target. For an indirect labelling, use "indirect".

popBH

character. The default is "alternate" : For the BH calculation, all the probe non expressed and non differentiel will exclude. The other choice is "complete" for the BH calculation on all the probes.

statBH

character. The default is "BH". Other choice are "Bonferroni", "Bonferroni.with.m0" and "qvalue".

bkgCalculation

logical. Default is TRUE. For calculation and substraction of the background. For the background calculation, a subset of the 10 percent less expressed probes is used for mean and standard deviation. The background is this mean plus twice standard deviation.

plot

logical. Default is TRUE, for generate the png quality plot.

adresse

character. The default is "http://pegasus-bioinfo.angers-nantes.inra.fr/rscripts/Tools/Agilent/", used for the specification of the design and the read of annotations file.

dataTest

logical. Default is FALSE.

dec

character. Default is "." as decimal separator.

Value

Summary of quality appreciations. Files are wrote on a new directory using the summary file name.

Prerequisites and summary file

The summary file must be in the same directory than raw data. It is a tabular text file with 7 columns and one line by array.

FileName

The summary file name of the array with the barcode of the slide and the position of the array on the slide. ei: 258455010001_1_1

Cy3

Name of the Cy3 sample. ei: NaCl_repet1

Cy5

Name of the Cy5 sample. ei. H2O_2

Control

3 or 5: the dye of the control sample (denominator for comparison)

Swaps

Name of the comparison. Must be the same for all the array (line) composing the comparison. ei: NaCl_vs_H2O

CtrName

Name of the control sample (denominator) after compilation (mean) of biological or thechnical repets.

TtmtName

Name of the treated sample (numerator) after compilation (mean) of biological or thechnical repets.

Note

For more explanation or problems, please contact the IRHS BIDeFI team : contact-bioinfo-irhs@inra.fr

Examples

anaDiff_Agilent(85550)
anaDiff_Agilent("85550")
anaDiff_Agilent("85550", labelling = "indirect", statBH = "Bonferroni")
